# Fetch-the-bone
Fetch the Bone is a fun browser-based game where you control a cartoon dog using your hand!
Move your hand in front of your laptopâ€™s camera, and the dog follows your motion in real time.
Catch as many falling bones as you can before your lives run out!
This project is built using HTML, CSS, JavaScript, and hand-tracking via webcam.
 Features:
 Hand-tracking control (no keyboard/mouse needed)
 Animated dog sprite that follows your hand
 Falling bones with increasing difficulty
 Lives, scoring & game over screen
 Restart button to play again
 Uses HTML5 Canvas for graphics
